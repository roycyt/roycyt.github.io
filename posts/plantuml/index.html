Play with the online server without installation.

Prerequisite:

$ sudo apt install default-jre graphviz
$ java -version 
openjdk version "11.0.8" 2020-07-14

Installation:

  • Download the PlantUML compiled Jar.

$ cat sequenceDiagram.txt

$ java -jar plantuml.jar sequenceDiagram.txt
$ xdg-open sequenceDiagram.png
